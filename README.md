 # Рыболовный Клуб — Установка форума bbPress

## О проекте
Полное пошаговое руководство по установке и настройке форума для сайта рыболовного клуба на WordPress с использованием плагина bbPress. 

## 🎯 Цель
Добавить на сайт работающее сообщество рыбаков, где пользователи могут общаться, задавать вопросы и делиться отчётами.

## 📋 Требования к форуму
- ✅ Минимум 1 форум
- ✅ Минимум 3 темы
- ✅ Минимум 3 сообщения
- ✅ Форум соответствует теме рыболовного клуба и общему стилю сайта
- ✅ Доступен через меню сайта
- ✅ Работает без ошибок

## 🚀 Инструкция по установке

### Шаг 1. Ручная установка bbPress
1. Скачайте плагин с официального сайта WordPress
2. Распакуйте архив и загрузите папку `bbpress` в `/wp-content/plugins/`
3. Активируйте плагин в админ-панели

### Шаг 2. Создание структуры форума
- **Рыболовный клуб** (главный форум)
- Категории: Отчёты о рыбалке, Советы и снасти, Где половить?

### Шаг 3. Добавление на сайт
- Создана страница со шорткодом `[bbp-forum-index]`
- Страница добавлена в главное меню навигации
- Обновлены постоянные ссылки
